# Prompt Library

These prompts are for capable coding agents with:

- shell access
- git access
- network access
- permission to read and modify the current repo

Recommended prompts:

- [`bootstrap-into-current-repo.md`](./bootstrap-into-current-repo.md)
  First-time install into a host repo.
- [`refresh-current-repo.md`](./refresh-current-repo.md)
  Re-run research and update the curriculum after the host repo changes.

The intended install shape is `./.vibe-coach/` inside the host repo.
